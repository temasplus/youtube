# youtube - Youtube TemasPlus (FREE)

TemasPlus is a free app for changing YouTube layout. The idea was to make a system of themes available on YouTube to the entire public, free of charge, without advertisements or additional costs.
- Website: www.temasplus.x10.mx

- App Free 
- Youtube TemasPlus Chrome: https://chromewebstore.google.com/detail/youtube-temasplus/ldiicanmmmikdmpdilnkghkphnancpno
- Youtube TemasPlus Firefox: https://addons.mozilla.org/firefox/addon/youtube-temasplus
